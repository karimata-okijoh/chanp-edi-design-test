# 既存データベース仕様書

## 概要

本文書は、MS-Accessで構築された既存アプリケーションのデータベース構造を定義します。
このデータベースから発注データを抽出し、共通EDI準拠のCSV形式で出力する機能を追加します。

## データベースファイル構成

### 1. common-data-option-chanp.mdb
業務データを管理するメインデータベース

### 2. common-data-sub.mdb
システム設定やマスタデータを管理するサブデータベース

---

## EDI CSV出力に必要なデータ

### 発注データの抽出元

EDI CSV出力機能では、以下のテーブルからデータを抽出します：

1. **材料マスタ**: 発注対象の材料情報
   - 材料CD（品目コード）
   - 材料名（品名）
   - 単価
   - 発注単位
   - 仕入先CD

2. **仕入先**: 発注先情報
   - 仕入先ID
   - 仕入先名
   - FAX番号（EDI送信先識別用）

3. **施設**: 配送先情報
   - 施設ID
   - 施設名

### データ抽出条件

- 発注しない = False（発注対象の材料のみ）
- 使用あり = True（使用中の材料のみ）
- 選択 = True（選択された材料のみ、UI操作による）

### 必須項目のマッピング

| EDI項目 | MS-Accessフィールド | テーブル |
|---------|-------------------|---------|
| 発注番号 | （自動生成） | - |
| 発注日 | （システム日付） | - |
| 品目コード | 材料CD | 材料マスタ |
| 品名 | 材料名 | 材料マスタ |
| 数量 | （計算値） | - |
| 単位 | 発注単位 | 材料マスタ |
| 単価 | 単価 | 材料マスタ |
| 金額 | （計算値：数量×単価） | - |
| 納期 | （計算値：発注日+リードタイム） | - |
| 仕入先コード | 仕入先CD | 材料マスタ |
| 仕入先名 | 仕入先名 | 仕入先 |
| 配送先コード | 施設ID | 施設 |
| 配送先名 | 施設名 | 施設 |

## 注意事項

1. **文字コード**: 既存DBはShift_JISで運用されている可能性が高いため、CSV出力時にUTF-8への変換が必要
2. **日付形式**: DateTimeフィールドはYYYYMMDD形式に変換
3. **数値形式**: Currency型は小数点以下の桁数に注意
4. **NULL値処理**: 必須項目がNULLの場合はエラーとして処理
5. **リレーション**: 材料マスタ.仕入先CD → 仕入先.仕入先ID の関連を考慮
